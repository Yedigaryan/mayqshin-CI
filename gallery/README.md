# Gallery Images

Please add your gallery images here. The gallery component expects images with the following names:
- project-1.jpg
- project-2.jpg
- project-3.jpg
- project-4.jpg
- project-5.jpg
- project-6.jpg

Or update the gallery array in `app/lib/content.ts` to match your image filenames.

Recommended format: JPG or WEBP, optimized for web (800x600px or similar aspect ratio).
